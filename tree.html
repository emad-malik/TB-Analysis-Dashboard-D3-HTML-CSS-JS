Tree Hierarchical Visualization

This visualization displays a tree-based hierarchical view of global tuberculosis (TB) incidence data, categorized by regions and countries. The size of each rectangle corresponds to the number of TB incidents in each country, while the color intensity represents the incidence rate per 100,000 people. You can select a specific region from the dropdown menu to filter the data and explore the TB incidence rates for different regions.
